Run the following command before running the app:
(Windows(/MacOS?) + Nvidia GPUs):
pip install -r nvidiarequirements.txt
(Windows(/MacOS?) + AMD GPUs):
pip install -r amdrequirements.txt
(Windows(/MacOS?) + ARM or Pine ):
pip install -r armorpinerequirements.txt
(Linux + Nvidia GPUs):
sudo pip install -r nvidiarequirements.txt
(Linux + AMD GPUs):
sudo pip install -r amdrequirements.txt
(Linux + ARM or Pine):
sudo pip install -r armorpinerequirements.txt
